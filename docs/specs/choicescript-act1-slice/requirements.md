# Requirements

Build a ChoiceScript demo that routes the full released Act 1 public corpus.

## User Experience

- Player begins at an Act 1 entry scene with title and Cessnya Lin byline.
- Player can follow the main Act 1 spine from prologue through chapter 20.
- Player can branch into every lettered optional Act 1 scene at the relevant
  chapter.
- Stats remain visible for the next mechanics pass.

## Constraints

- Use native ChoiceScript scene files.
- Run quicktest and randomtest in the release gate.
- No later-act material.
- No dependency on Twine, Ren'Py, or Godot code.
