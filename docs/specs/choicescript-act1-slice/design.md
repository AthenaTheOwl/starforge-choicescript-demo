# Design

## Scenes

- `startup.txt`: title, author, stat creation, scene list, initial choice.
- `ch_*.txt`: generated main and optional Act 1 scenes.
- `act1_ending.txt`: route-complete ending.

## Stats

- `resolve`
- `empathy`
- `tech`
- `lattice`
- `crew_trust`
- `scenes_seen`

## Checks

- P0 checks are route/reachability checks, not tuned stat-gated mechanics.
- Each main chapter exposes its optional lettered scenes before continuing.
- Each optional scene can continue to the next optional sibling or rejoin the
  main spine.
